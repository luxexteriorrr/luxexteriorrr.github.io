Vestige Manual

Vestige is an interactive, web-driven installation exploring Internet Decay and Digital Preservation. The project immerses the viewers in a metaphorical digital environment where interactions with the installation trigger the rate of the enviroment’s deterioration. The installation encapsulates a calculated timeline of the webpage decay, altering the readability, positionality, and the very existence of HTML elements, culminating in a unsusable/empty HTML document.

this project is about internet decay. to explore, scroll

In front of you is a browser-based component of an ongoing project exploring ambience, digital decay and narrative erosion. At the bottom of the screen, there is a timestamp providing the last update on the project.

The textures, scripts, visual residues were drawn from experience of digital and physical ambience. Thus the moment of such is simulated through this experience. The page echoes what is remembered, more so what is rendered — an interface less for navigation but more so of a presence.

It is a system of eroding containers, inspired by internet decay and its manifestations. The interface is a digital publication which unfolds with your interactions, resisting and decaying in real time.

There are many pathways and hidden details in the following interface. Explore through scroll, click and time. Elements will unfold as you scroll, unravelling the narrative. Take time to read them and interact with them in order to keep reading.
